### 💻 Пример кода (OpenAI)

Вот как выглядит простой чат-бот на Python с использованием библиотеки OpenAI. Этот код создает клиент, отправляет список сообщений и выводит ответ от ИИ.

```python
import os
from openai import OpenAI

# --- Инициализация клиента (API ключ берется из переменных окружения) ---
client = OpenAI(api_key=os.getenv("OPENAI_API_KEY"))

# --- Список для хранения истории сообщений (используем для долгой памяти) ---
conversation_history = [
    {"role": "system", "content": "Ты — полезный ИИ-ассистент."}
]

print("Чат-бот запущен. Введите 'выход' для завершения.")
while True:
    user_input = input("Вы: ")
    if user_input.lower() == 'выход':
        break

    # Добавляем сообщение пользователя в историю
    conversation_history.append({"role": "user", "content": user_input})

    # Отправляем запрос к API
    response = client.chat.completions.create(
        model="gpt-4o-mini",  # Экономичная модель
        messages=conversation_history,
        temperature=0.7  # Контролирует креативность ответов
    )

    # Получаем ответ ассистента
    assistant_response = response.choices[0].message.content

    # Добавляем ответ ассистента в историю
    conversation_history.append({"role": "assistant", "content": assistant_response})

    print(f"Бот: {assistant_response}")
```

### ⚙️ Как установить плагины в Notepad++

Установка обычно не вызывает сложностей. Вот самые частые способы:
*   **Через Plugin Admin**: это самый простой метод. Откройте `Plugins` → `Plugins Admin...` и установите галочку напротив нужного плагина, чтобы установить его. Если интересующего вас плагина там нет, можно добавить его вручную.

*   **Настройка API**: практически все AI-плагины требуют ввода **API-ключа** от соответствующего сервиса. Ключ нужно будет вставить в конфигурационный файл или в окно настроек плагина.

### ⚙️ Пошаговая настройка

1.  **Убедитесь, что у вас актуальная версия плагина**
    Этот шаг очень важен, так как старые версии могут не поддерживать длинные ключи API или возможность смены API-адреса, которая нам понадобится далее[reference:0][reference:1].
      * **Что нужно сделать:** Обновите плагин доверсии **v0.4.1 или новее**. Рекомендуется установить последнюю версию (например, v0.5.0), которая точно поддерживает все необходимые функции[reference:2][reference:3].
      * **Как обновить:** Вы можете попробовать обновить плагин через встроенный менеджер плагинов Notepad++ (`Plugins` -> `Plugins Admin...`). Если там нет новой версии, скачайте её вручную с [официальной страницы на GitHub](https://github.com/Krazal/nppopenai) и замените файлы в папке плагинов[reference:4].

2.  **Получите API-ключ DeepSeek**
    Это ваш личный идентификатор для доступа к API DeepSeek[reference:5].
      * **Где взять:** Перейдите на [платформу DeepSeek](https://platform.deepseek.com/), войдите в аккаунт и в разделе "API Keys" нажмите кнопку "Create New Key" или "Создать новый ключ"[reference:6].
      * **Важно:** Скопируйте и сохраните ключ сразу после создания — он отображается только один раз[reference:7].

3.  **Настройте конфигурационный файл `NppOpenAI.ini`**
    Это сердце всей настройки, где мы укажем DeepSeek в качестве "провайдера" для плагина[reference:8].
      * **Как открыть файл:** В Notepad++ перейдите в меню `Plugins` -> `NppOpenAI` -> `Edit Config`[reference:9].
      * **Что нужно изменить:** В открывшемся файле найдите секцию `[API]` и приведите её к следующему виду, подставив свой ключ:

    ```ini
    [API]
    ; Это самый важный параметр — ваш секретный ключ от DeepSeek
    secret_key=sk-...

    ; URL API-адреса DeepSeek. Для работы через прокси в РФ используйте другой адрес (см. пункт 4)
    api_url=https://api.deepseek.com/v1/chat/completions

    ; Название модели DeepSeek
    model=deepseek-chat

    ; Остальные параметры можно оставить без изменений или настроить по желанию
    temperature=0.7
    max_tokens=0
    top_p=0.8
    frequency_penalty=0
    presence_penalty=0

    [PLUGIN]
    ; Эта опция оставляет ваш исходный вопрос в редакторе после получения ответа
    keep_question=1
    ```

4.  **Используйте прокси (если это необходимо)**
    Если вы находитесь в РФ и прямой доступ к API DeepSeek по адресу `api.deepseek.com` у вас отсутствует, потребуется настроить прокси.
      * **Как настроить:** Отредактируйте параметр `api_url`. Вместо стандартного адреса провайдера, вам нужно будет указать адрес прокси-сервера, который будет перенаправлять запросы к DeepSeek[reference:10][reference:11]. Точный адрес зависит от вашего прокси-провайдера.

5.  **Сохраните и перезагрузите конфигурацию**
    После внесения всех изменений сохраните файл (`Ctrl+S`).
    Затем вернитесь в меню `Plugins` -> `NppOpenAI` и выберите **`Load Config`**, чтобы плагин применил новые настройки без перезапуска Notepad++[reference:12]. Можно также просто перезапустить редактор[reference:13].

### ⚠️ Возможные проблемы и их решение

*   **Ошибка "Incorrect API key"**: Убедитесь, что скопировали ключ целиком и без лишних пробелов. Если ключ очень длинный (более 50 символов) и вы используете старый плагин, это — основная причина ошибки. Обновите плагин до последней версии, как указано в шаге 1[reference:15][reference:16].
*   **Ничего не происходит или ошибка сети**: Проверьте, правильно ли указан параметр `api_url`. Если вы настроили прокси, убедитесь, что его адрес верен и он работает. Также проверьте интернет-соединение.
*   **Неверное имя модели**: Если API возвращает ошибку, что модель не найдена, проверьте, точно ли в `model` указано `deepseek-chat`. Это стандартное имя для текстовой модели DeepSeek[reference:17].

### 💡 Советы для эффективной работы

*   **Работайте с инструкциями**: Вы можете задать для плагина контекст или роль. Для этого откройте файл с инструкцией через `Plugins` -> `NppOpenAI` -> `Edit Instructions`. Всё, что вы там напишете, будет влиять на ответы AI. Например, можно прописать там: `Ты — опытный Python-разработчик`, и ответы будут даваться в этом ключе[reference:18].
*   **Используйте горячие клавиши**: Основное сочетание `Ctrl+Shift+O` можно дополнять другими для ускорения работы: `Ctrl+V` (вставить), `Ctrl+A` (выделить всё), `Ctrl+C` (скопировать). Это позволяет быстро исправлять готовый код или анализировать текст[reference:19].
