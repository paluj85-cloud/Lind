      }

  }

### 3. Файл `_validation_test.md`
Отчёт о проверке точности. Для КАЖДОГО факта:
```markdown
## Факт test-v-001
- **Утверждение**: [fact]
- **Цитата**: [source_quote]
- **Вердикт**: ✅ Подтверждено | ❌ Не подтверждено
- **Комментарий**: [если не подтверждено — что не так]
```

В конце файла:
```markdown
## Итого
- Всего фактов: N
- Подтверждено: M (X%)
- Не подтверждено: K (Y%)
- Цель: ≥95%
- Статус: ПРОЙДЕН | НЕ ПРОЙДЕН
```
