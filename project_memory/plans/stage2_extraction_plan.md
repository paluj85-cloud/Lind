## На каждой партии
1. Прочитать 10 чатов из `chat_deepseek/`
2. Извлечь факты в JSON: `{fact, source_quote, section, confidence}`
3. Сразу записать в `adnd-knowledge/{section}/_audit.json`
4. Валидировать 15% фактов (≥95%) через grep `source_quote`
5. Чекпоинт: ✓ — едем дальше, ✗ — корректируем и повторяем

## Структура выхода

├── _audit.json        ← [факт, source_quote, chat_id, confidence]
├── _source_map.json   ← [fact_id → chat_id, line_range]
