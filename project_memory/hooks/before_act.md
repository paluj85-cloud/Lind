# Before Act Hook

Run this checklist before making ANY code changes.

## Checklist
- [ ] Mode is ACT (not Plan)
- [ ] Prompt is read and understood
- [ ] Context is loaded (relevant files, DB schema, APIs)
- [ ] Constraints are clear (tech stack, no hardcoded secrets, etc.)
- [ ] Plan is understood (what to build, in what order)
- [ ] Acceptance criteria are defined

## If anything is unclear → STOP and ask the user.
