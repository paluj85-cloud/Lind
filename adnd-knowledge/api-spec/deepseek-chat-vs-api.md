# DeepSeek: Чат vs API

**Дата извлечения**: 2026-05-16
**Источник**: 2026-04-21/e3741e54-сравнение-чата-и-api-deepseek.md
**Раздел**: api-spec

## Суть
Сравнение двух способов взаимодействия с DeepSeek: бесплатный веб-чат и платное API. Чат — для работы с текстом/документами, API — для интеграции в свои приложения.

## Ключевые факты

### Чат DeepSeek (бесплатно)
- Контекст: 1M токенов (трилогия «Трёхтелье»)
- Бесплатно, без квот
- Загрузка файлов до 100 МБ (PDF/Word/Excel/изображения)
- Поиск в интернете — одна кнопка
- Голосовой ввод
- Не требует навыков программирования
- **Нельзя** автоматизировать или встроить в свой проект

### API DeepSeek (платно, дёшево)
- Позволяет встроить ИИ в свои программы, сайты, чат-боты, CRM
- Тонкая настройка: `temperature`, `top_p`, `max_tokens`
- Параллельные запросы (сотни/тысячи)
- Streaming (печать по мере генерации)
- Контекст до 1M (зависит от тарифа)
- Требует навыков программирования
- Поиск в интернете нужно настраивать отдельно
