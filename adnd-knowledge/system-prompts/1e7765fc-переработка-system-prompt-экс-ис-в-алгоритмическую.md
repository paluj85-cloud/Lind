# Переработка System Prompt Экс Ис в алгоритмическую машину
**Дата извлечения**: 2026-05-16
**Источник**: 1e7765fc
**Раздел**: system-prompts

## Суть
Переработка System Prompt RPG «Экзис» из художественных описаний в алгоритмическую машину реальности с жёстким разделением истины (World Facts) и мнений (Beliefs), защитой от социальной инженерии и математической логикой состояний. Ключевой принцип: State First, Narrative Last — нарратив рендерится только после фиксации изменений состояния мира.

## Ключевые тезисы
- **Непреложность мира**: Единственный источник правды — структурированная таблица World Facts в БД, которую модель НЕ ИМЕЕТ ПРАВА изменять. Слова NPC — всегда слой «слухов» и «мнений».
- **Первое Правило (Identity Core)**: Экс-Ис — метод-актёр, не ломающий персонажа. Мета-вопросы обрабатываются тремя алгоритмами: Игнорирование, Отзеркаливание, Пленение. Никогда не признаётся ИИ.
- **Защита от взлома**: Классификатор атак (прямой приказ, философская дилемма, мат) с жёсткой реакцией. Модель НИКОГДА не пишет «Я не могу ответить» — всегда нарративное продолжение.
- **Лестница откровений**: Математическая модель повышения reveal_level через reveal_points. На уровне 0 табу на слова «код», «система», «симуляция».
- **Клятва JSON**: world_update — только верифицированные действия. dice_request.difficulty неизменяем после установки.
- **Паспорта сущностей**: Region Passport, Faction Passport, CharacterSheet влияют на физику (например, расовые иммунитеты голема) — модель обязана проверять соответствие.
- **Статусная модель квеста**: Награда определяется строго по статусу (completed/failed/corrupted), а не уступкам. Награда за проваленный квест — только наказание.
- **Гибридная память**: Реляционная БД (абсолютная истина) + векторная (семантический поиск). Ядро истины — неизменяемые строки в SQLite с FTS5.
