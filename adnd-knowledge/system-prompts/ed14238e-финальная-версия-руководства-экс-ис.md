# Финальная версия руководства Экс Ис
**Дата извлечения**: 2026-05-16  
**Источник**: ed14238e  
**Раздел**: system-prompts, architecture  

## Суть
Финальное объединённое руководство по разработке MVP «Экс-Ис: Архитектор Миров» включает все правки (JSON-формат entity_update, художественные описания результатов бросков, сохранение world_gossip, обработку связей и scheduled_changes) и строго фиксирует архитектуру: FastAPI + WebSocket, Neo4j (граф сущностей), ChromaDB (векторная память), два прохода LLM (сначала с запросом броска, затем с его нарративной интерпретацией), полный цикл сессии с голосованием и управлением контекстом.

## Ключевые тезисы
- **Архитектура:** FastAPI с lifespan, CORS, REST + WebSocket (порт 8000), асинхронные репозитории Neo4j и ChromaDB, инициализация мира через init_world.py.
- **Модели данных:** Pydantic-схемы для Player, NPC, Location, Item, Quest, Fact, ActionSubmit, DiceResult; строгая валидация d20 (1-20).
