## Архитектура промптов (v5.0)

```
РИТУАЛ ПЕРВОГО КОНТАКТА (greeting.md)
       │
       ▼
СОТВОРЕНИЕ МИРА (world-creation.md) — однократно при старте кампании
       │
       ▼
ИГРОВОЙ ЦИКЛ (exis-master.md) — каждый ход, 6 блоков:
   1. <narrative>           — что видит игрок
   2. <dice_request> / <anima_response> — проверки и совет
   3. <state_update>        — изменения публичной истины
   4. <secret_memory>       — скрытая истина
   5. <service_update>      — entity, слухи, квесты, world_pulse
   6. <context_request>     — запрос контекста на следующий ход
```
