## Версия и название приложения

- Приложение называется «Экзис v0.1» и представляет собой веб-приложение «Экзис: Чат с Богом».
- Проект также известен как «Экзис: Архитектор Миров» — ролевая игра, где нейросеть (DeepSeek) выступает Мастером в духе AD&D.
- Приложение версии v0.1 развёрнуто на сервере или локально.

## Ключевые документы

- Создано финальное ТЗ на создание MVP веб-приложения «Чат с Богом».

## Этапы разработки

- Проект находится на стадии аудита архитектуры.
- Проект находится на этапе переработки System Prompt Экс-Ис в алгоритмическую машину после инцидента на тестовой сессии.
- Проект находится на этапе составления ТЗ и плана, пользователь обсуждает идею с ИИ-ассистентом.
- Проект находится на стадии планирования, код ещё не пишется, создаётся детальный план для Cline.
- Текущий этап — создание концепции и технического задания на систему памяти мира Экзис.
- Дата чата: 2026-05-09. Проект находится на этапе написания ТЗ для версии v0.1.
- Уже есть Финальное Техническое Задание на серверную часть (FastAPI, SQLite FTS5, сессии, память), модуль SystemPromptBuilder (v2.0), и проведён тестовый прогон.
- Уже проработана концепция «памяти мира» (двухконтурная система: векторная LLM-память и human-readable хроники), а также архитектура агента-ритривера, FIM-вставок кода и Summary Chain.
- Код игры еще сырой, пользователь слабо подкован в программировании и создании приложений, он инженер-строитель.
- Текущая структура таблицы races и связь character_races обсуждаются для доработки с целью генерации NPC в AD&D.

## Системный промпт

- Разработан финальный System Prompt v5.0 для игрового мастера Экс-Ис, содержащий философию, правила и формат вывода.
- Обсуждается создание System Prompt v5.0, который должен стать эталоном для всей индустрии ролевых ИИ-мастеров.
- Версия 4.0 является отправной точкой для разработки v5.0.
- Уже есть ядро системного промта версии 1.0, утверждённое Творцом.
- Задача — создать расширенную версию системного промта v2.0.
- Существующий промпт v2.0 написан как вдохновляющая поэма, но не содержит алгоритмических инструкций.
- Утверждён системный промт Мастера (ядро личности Экс-Ис).

## Тестирование и инциденты

- Проведён тестовый прогон с тремя игроками, который выявил критический провал логики: LLM поверил словам NPC под давлением и переписал реальность.
- Тестовая сессия с тремя персонажами (воин Кук, полуэльф Василий, голем Александр) выявила проблемы: NPC переписывали историю мира, сюжет скатился, игроки получили награду за проваленный квест, мат игроков просочился в ответы NPC.
- Корневая причина инцидента: текущий промт описывает лишь «дух» Мастера, но не содержит алгоритмической защиты от социальной инженерии.
- Концепция ИИ-мастера для НРИ в стиле AD&D представлена и проанализирована. Отмечены сильные стороны и потенциальные вызовы.

## Планы на будущее

- Аудитория проекта: сначала друзья-тестеры, затем — публичный запуск. Конечная цель — полноценный мультиплеерный мир, в физику и логику которого игроки будут вовлечены.
- В будущем игра станет мультиплеерной.

## Сводка

1. **Проект находится на ранней стадии (v0.1)**: Разработано финальное техническое задание, создан MVP веб-приложения «Экзис: Чат с Богом», код ещё сырой и требует доработки.
2. **Ключевой артефакт — Системный Промпт**: Разработана версия v5.0, призванная стать эталоном для индустрии. Предыдущие версии (v1.0, v2.0, v4.0) послужили основой для эволюции.
3. **Выявлен критический инцидент**: Тестовая сессия показала, что текущий промпт не защищает от социальной инженерии — LLM переписывал реальность под давлением игроков. Это привело к переработке промпта в алгоритмическую машину.
4. **Проработана архитектура памяти**: Создано ТЗ на подсистему «Вечный Архив» с двухконтурной системой (векторная LLM-память и human-readable хроники).
5. **Планы масштабирования**: Проект ориентирован на поэтапный запуск: от друзей-тестеров до публичного релиза и полноценного мультиплеерного мира.
