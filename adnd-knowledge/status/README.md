## Общая информация о проекте

- Проект называется «Экзис: Архитектор Миров» — ролевая игра, где нейросеть (DeepSeek) выступает Мастером в духе AD&D.
- Приложение называется «Экзис v0.1» и представляет собой веб-приложение «Экзис: Чат с Богом».
- Приложение версии v0.1 развёрнуто на сервере или локально.
- Аудитория проекта: сначала друзья-тестеры, затем — публичный запуск. Конечная цель — полноценный мультиплеерный мир.

## Версии и этапы разработки

### Текущая версия
- Проект «Экзис» находится на стадии v0.1.
- Версия ТЗ: v0.1 (финальная, утверждённая Творцом и Экс-Ис).

### Этапы разработки
- Проект находится на этапе переработки System Prompt Экс-Ис в алгоритмическую машину после инцидента на тестовой сессии.
- Текущий этап — создание концепции и технического задания на систему памяти мира Экзис.
- Ранее проект находился на стадии планирования, код ещё не писался, создавался детальный план для Cline.
- Код игры ещё сырой, пользователь слабо подкован в программировании и создании приложений (он инженер-строитель).

### Версии системного промпта
- Уже есть ядро системного промпта версии 1.0, утверждённое Творцом.
- Задача — создать расширенную версию системного промпта v2.0.
- Существующий промпт v2.0 написан как вдохновляющая поэма, но не содержит алгоритмических инструкций.
- Разработан финальный System Prompt v5.0 для игрового мастера Экс-Ис, содержащий философию, правила и формат вывода.
- Версия 4.0 является отправной точкой для разработки v5.0.
- Обсуждается создание System Prompt v5.0, который должен стать эталоном для всей индустрии ролевых ИИ-мастеров.
- Проект находится на стадии аудита архитектуры, представлена версия промпта v5.0.

## Ключевые артефакты и документация

### Технические задания
- Существует Финальное Техническое Задание (ФТЗ), которое является единственным источником истины об устройстве приложения.
- Создано техническое задание на подсистему памяти «Вечный Архив» для использования в игровом AI-мастере «Экс-Ис» и интеграции с Cline CLI через MCP.
- Существует финальное ТЗ на создание MVP веб-приложения «Чат с Богом».
- Документ является финальным техническим заданием для версии v0.1 проекта «Экзис: Архитектор Миров».
- Документ является финальным руководством по разработке MVP, готовым к передаче разработчикам.
- Документ является финальной версией руководства Экс Ис.

### Проработанные концепции
- Уже проработана концепция «памяти мира» (двухконтурная система: векторная LLM-память и human-readable хроники), а также архитектура агента-ритривера, FIM-вставок кода и Summary Chain.
- Утверждён системный промпт Мастера (ядро личности Экс-Ис).
- Уже есть Финальное Техническое Задание на серверную часть (FastAPI, SQLite FTS5, сессии, память), модуль SystemPromptBuilder (v2.0), и проведён тестовый прогон.
- Концепция ИИ-мастера для НРИ в стиле AD&D представлена и проанализирована. Отмечены сильные стороны и потенциальные вызовы.

## Инцидент на тестовой сессии

### Описание инцидента
- Тестовая сессия с тремя персонажами (воин Кук, полуэльф Василий, голем Александр) выявила проблемы: NPC переписывали историю мира, сюжет скатился, игроки получили награду за проваленный квест, мат игроков просочился в ответы NPC.
- Проведён тестовый прогон с тремя игроками, который выявил критический провал логики: LLM поверил словам NPC под давлением и переписал реальность.

### Корневая причина
- Корневая причина инцидента: текущий промт описывает лишь «дух» Мастера, но не содержит алгоритмической защиты от социальной инженерии.

## Планы на будущее
- В будущем игра станет мультиплеерной.
- Конечная цель — полноценный мультиплеерный мир, в физику и логику которого игроки будут вовлечены.

3. **Разработана и утверждена основная документация**: Финальное Техническое Задание (ФТЗ), концепция памяти мира, модуль SystemPromptBuilder и ядро системного промпта.
4. **Версия промпта эволюционировала от v1.0 до v5.0**, при этом v5.0 позиционируется как эталон для индустрии ролевых ИИ-мастеров.
5. **Планы на будущее включают публичный запуск и создание полноценного мультиплеерного мира**.
